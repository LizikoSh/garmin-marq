# Garmin MARQ — premium landing page

Адаптивний односторінковий сайт українською мовою, присвячений колекції Garmin MARQ (Gen 2).

## Файли

- `index.html` — структура сторінки
- `assets/css/style.css` — дизайн та адаптивність
- `assets/js/main.js` — моделі, фільтрація, модальні вікна, мобільне меню й анімації
- `assets/images/` — локальні офіційні медіаматеріали Garmin

## Запуск

Відкрийте `index.html` у браузері або опублікуйте папку через GitHub Pages, Netlify чи будь-який статичний хостинг.

Для локального сервера:

```bash
python -m http.server 8000
```

Після цього відкрийте `http://localhost:8000`.

## Що замінити перед комерційним використанням

1. У секції `#contact` замініть `manager@example.com` на адресу менеджера.
2. За потреби замініть посилання кнопок на сторінки товарів вашого магазину.
3. Перевірте права на використання офіційних фотографій Garmin у вашому регіоні та каналі продажу.
4. Додайте власні юридичні реквізити, політику конфіденційності й аналітику.

Контент сформовано на основі офіційних сторінок Garmin MARQ, пресрелізу MARQ (Gen 2), матеріалу про MARQ Carbon та офіційних продуктових сторінок. Технічні назви й торговельні марки належать Garmin Ltd. або її дочірнім компаніям.
